# Telco Customer Churn (Retención de clientes)

He decidido empezar un proyecto personal para aprender técnicas y workflows de data science. 
Todo este proyecto tiene un enfoque en aprender conceptos nuevos y afianzar buenas prácticas.
Además, he aprovechado este proyecto para introducirme en el uso de Docker, entendiendo cómo crear entornos reproducibles, levantar contenedores y gestionar dependencias de forma aislada.

## Objetivos del proyecto
- Comprender el problema de churn.
- Explorar el dataset de Telco Customer Churn.
- Aplicar técnicas de preprocesamiento.
- Entrenar modelos supervisados y evaluar resultados.
- Documentar el flujo de trabajo.

## Plan de trabajo
1. Carga y exploración de datos
2. Limpieza y preparación
3. Análisis exploratorio (EDA)
4. Modelado
5. Validación
6. Conclusiones
